# 20分钟船游戏项目

这是一个小游戏项目，旨在锻炼游戏设计的能力。因此选用[Godot](https://godotengine.org/)引擎实现。

## 项目的特点

* 开放。大家都可以参与讨论。
* 文档驱动。设计文档均放在`docs/`目录下面。

## 游戏的核心目标

核心目标是这个游戏的一切，应属于最稳定的部分，没有十分充足的理由，可以说是不可改变的。
设计应当为了满足核心目标而出现，不应背道而驰。

本游戏的核心目标为：

1. 单局时长15分钟到30分钟，希望能够被反复游玩。
2. 一局游戏内，希望玩家的爽感，心率等指标能逐步提升，在游戏终局达到顶峰，获取快乐。

## 工程进行方式

* 设计

1. 在`docs/drafts/`目录中新建一个草案，联系组内人员进行讨论。
组内人员人数在1人时，可免去这个步骤。

2. 讨论通过的草案，直接写入文档中`docs/*.md`

* 开发

通过的草案，即文档中的内容，可以直接开发至工程中。
